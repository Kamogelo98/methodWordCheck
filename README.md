# methodWordCheck

This is a function that takes in a word and returns a string "long" if the word is longer than 6 characters ,"short" if it is less than 6 characters and "medium" if it is exactly 6 characters

## Built With

* Visual Studio Code
* HTML
* JavaScript

## Tested With
Live Server

### Prerequisites:
